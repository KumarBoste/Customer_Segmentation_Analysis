# Customer Segmentation Analysis ( Excel )
![Dashboard](https://github.com/KumarBoste/Customer_Segmentation_Analysis/blob/main/Excel%20Dashboard/Customer%20Segmentation%20Dashboard.png)

## Project Objective
The primary objective of this project is to analyze and segment customers based on demographics, income, tenure, profession, loyalty, and geographic regions.
The dashboard helps businesses understand customer behavior, identify high-value segments, and make data-driven marketing and engagement decisions.

## Problem Definition
Organizations often struggle to:
- Identify valuable customer segments
- Understand customer loyalty and wealth distribution
- Personalize marketing strategies across regions and customer types
- Track customer growth trends over time
This dashboard addresses these challenges by providing interactive visual insights.

## Key Business Questions
The dashboard answers the following questions:
1. How many total customers does the organization have?
2. What is the average age, income, and tenure of customers?
3. Which generation segment (Boomers, Gen X, Gen Z, Millennials) dominates?
4. What percentage of customers are loyal vs new or emerging?
5. How is customer wealth distributed (Low, Medium, High)?
6. How has the customer base grown over time?
7. What professions contribute most to the customer base?
8. How do regions and customer types affect customer distribution?

## Dataset Description
The dataset includes customer-related attributes such as:
- Customer count
- Age
- Income
- Tenure (years)
- Gender
- Generation group
- Loyalty level
- Wealth category
- Profession
- Region
